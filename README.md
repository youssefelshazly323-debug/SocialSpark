# SocialSpark

## About
This application recommends basic social events that depend on the time of the day.

## How to Use
1. Time of day in the input box Type a
2. Press "Get Social Spark"
3. Follow the suggestion!

## Valid Inputs
- Morning
- Mid-Morning
- Afternoon
- Afternoon Snack Time
- Dinner
- After Dinner
